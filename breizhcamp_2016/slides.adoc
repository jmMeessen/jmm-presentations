== Les dangers avec Docker ?

[%step]
* Kernel exploits
* Denial of service attack
* Container breakout
* Poisoned images
* Compromising Secrets

== Est-ce que Docker est "secure" ?

* Beaucoup d'attentes, d'illusions
* "Silver bullet"
* positionement de concurents (VM, Configuration Mgt)
* jalousie
